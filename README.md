# LEARN
### Opis algorytmu:

1. Program pyta użytkownika o nazwę produktu i liczbę miesięcy.
2. Dla każdego miesiąca użytkownik wprowadza dane sprzedaży.
3. Dane są zapisywane w obiekcie `SalesData`.
4. Na ich podstawie obliczane są podstawowe statystyki: suma, średnia, min, max.
5. Tworzony jest wykres sprzedaży w czasie z adnotacjami min/max.
6. Użytkownik ma możliwość zapisania wykresu jako pliku graficznego.
